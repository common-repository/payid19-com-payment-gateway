== Description ==
-Crypto Payment Gateway you can accept USDT, Bitcoin, Litecoin, Ethereum, Bnb and TRX stable coins and withdraw as USDT.

-No chargebacks

- This is a WooCommerce module developed by [Payid19.com](https://payid19.com/) Integration team. 

- After the integration you will be able to use pay with Payid19.com automatically

- You can easily integrate the module into your e-commerce website and start receiving payment with cryptocurrencies seamlessly and securely. 

- You can embed Payid19.com check-out form either as responsive or pop-up onto your payment screen and increase your sales conversion.

- The module controlled by our team and it is totally secure.

- With the open source code, you can make easily make new developments on your website. 

- You can test your website in Test mode, one of the best test environments.

- Payid19.com module allows you to sell with installments on your website with supported cryptocurrencies.

- Payid19.Com Payment Gateway module is 100% compatible with WooCommerce and Wordpress systems. 

- After the integration you can offer manually created orders and payment support to your customers.

-Visit our [FAQ Page](https://payid19.com/faq) for other details.  

Requirements
PHP 5.4 and greater.
WooCommerce 3.5 requires WordPress 3.5+


== Installation ==
Payid19.Com Payment Gateway Plugin Download ZIP file and then Install with wordpress extension installer page

Wordpress Marketplace in search "Payid19.Com Crypto Payment Gateway" and install
